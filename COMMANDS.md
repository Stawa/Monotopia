# Command Checklist

Commands are loaded from `apps/server/src/command/cmds` and `apps/server/src/command/cmds/emotes`.

Permissions are hierarchical:

- Regular User can use Regular commands.
- Supporter can use Regular + Supporter commands.
- Moderator can use Regular + Supporter commands.
- Developer can use all commands.
